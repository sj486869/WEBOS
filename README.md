## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Go to the project directory:

```bash
cd your-repository
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser:

```
http://localhost:3000
```
